# Wazuh → Shuffle Integration

## Objective

Integrate Wazuh with Shuffle SOAR to automatically forward security alerts to an automation workflow using a webhook.

---

## Why Integrate Wazuh with Shuffle?

Wazuh is responsible for detecting security events, while Shuffle automates the response process.

Instead of manually investigating every alert, Wazuh forwards selected alerts to Shuffle, where automated workflows can enrich the incident and perform additional actions.

---

## Integration Workflow

```
Windows Endpoint
        │
        ▼
Wazuh Agent
        │
        ▼
Wazuh Manager
        │
        ▼
Webhook
        │
        ▼
Shuffle Workflow
```

---

## Configuration Steps

### 1. Copy the Shuffle Webhook URL

Open the Shuffle workflow.

Copy the webhook URL generated by the Webhook trigger.

---

### 2. Configure Wazuh

Open the Wazuh configuration file.

```
/var/ossec/etc/ossec.conf
```

Add a new integration section using the Shuffle webhook URL.

---

### 3. Restart the Wazuh Manager

Apply the configuration changes.

```bash
sudo systemctl restart wazuh-manager
```

---

## Validation

Generate a security alert that matches **Rule ID 100002**.

Verify that:

- Wazuh successfully forwards the alert.
- Shuffle receives the webhook.
- The workflow starts automatically.

---

## Result

Wazuh was successfully integrated with Shuffle, allowing security alerts to automatically trigger SOAR workflows.
